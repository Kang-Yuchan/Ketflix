# Ketflix

ReactとES6を勉強しながら、Netflixのようなアプリを制作(Using The Movie Database API)

## Screens

- [x] Home
- [x] TV Shows
- [x] Search
- [x] Detail

## API Verbs

- [x] Now playing (Movie)
- [x] Top Rated (TV)
- [x] Popular (TV, Movie)
- [x] Upcoming (Movie)
- [x] Airing Today(TV)
- [x] TV Show Detail
- [x] Movie Detail
- [x] Search (Movie, TV)
